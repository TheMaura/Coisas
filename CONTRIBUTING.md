# Guia de Contribuição - Futebol Legends

## Estrutura do Projeto

- `app/` - Telas do aplicativo usando Expo Router
- `contexts/` - Contextos React para estado global
- `lib/` - Bibliotecas e configurações (Supabase, etc.)
- `types/` - Definições TypeScript
- `supabase/` - Scripts SQL do banco de dados

## Convenções de Código

- Use TypeScript para type safety
- Siga os padrões do Expo Router
- Use componentes funcionais com hooks
- Mantenha componentes pequenos e reutilizáveis

## Fluxo de Desenvolvimento

1. Criar branch para nova feature
2. Desenvolver e testar localmente
3. Fazer commit com mensagens descritivas
4. Criar pull request

## Testes

Teste em:
- Android (emulador e dispositivo físico)
- iOS (se disponível)
- Web (opcional)
