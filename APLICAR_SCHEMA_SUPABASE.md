# Como Aplicar o Schema Estendido no Supabase

## ⚠️ IMPORTANTE

Antes de usar todas as funcionalidades, você precisa aplicar o schema SQL no Supabase.

## Passos:

1. **Acesse o Supabase Dashboard**: https://supabase.com/dashboard
2. **Vá para o projeto**: `lrkqhubivgozjkcdbisg`
3. **Navegue até SQL Editor**
4. **Cole o conteúdo do arquivo**: `supabase/schema_extended.sql`
5. **Execute o SQL**

## O que o schema adiciona:

- ✅ Campos estendidos em `profiles` (bio, stats, preferences)
- ✅ Tabela `stories` (histórias inspiradoras)
- ✅ Tabela `media` (imagens e vídeos)
- ✅ Tabela `comments` (comentários)
- ✅ Tabela `content_moderation` (moderação)
- ✅ Tabela `view_stats` (estatísticas)
- ✅ RLS policies para segurança
- ✅ Triggers para atualização automática

## Após aplicar:

Todas as funcionalidades do app estarão disponíveis!
